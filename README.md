# Sanad Crypto Tracker - Web3 AR/EN

A student full-stack crypto market project built with React.js, HTML, CSS, JavaScript, and Python Flask.

## New update
- The market table now works like a CoinGecko-style list.
- It loads 100 coins at a time from CoinGecko using pagination.
- Click **Load More Coins** to load more market pages.
- Search works inside the loaded coins.
- Arabic / English support and RTL are included.

## Run Backend
```bash
cd backend
pip install -r requirements.txt
python app.py
```

Backend URL:
```text
http://127.0.0.1:5000
```

## Run Frontend
Open a second terminal:
```bash
cd frontend
npm install
npm run dev
```

Frontend URL:
```text
http://localhost:5173
```
